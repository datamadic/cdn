Tutorial: system.getMonitorInfo

system.getMonitorInfo

Retrieves an object that contains data about the about the monitor setup of the computer that the runtime is running on.

Example

fin.desktop.System.getMonitorInfo(function (monitorInfo) {
    console.log("This object contains information about all monitors: ", monitorInfo);
});

Monitor Info
